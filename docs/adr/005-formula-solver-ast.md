# ADR-005: AST-парсер для формул Revit (универсальный модуль)

**Статус:** implemented
**Дата:** 2026-03-25
**Реализовано:** 2026-04-06

## Контекст

Размеры коннекторов в семействах Revit часто вычисляются через формулы:
```
Radius = if(DN < 50mm, DN / 2 - 1mm, DN / 2 - 2mm)
```

Для подбора параметра при несовпадении размеров нужно:
1. **Прямое вычисление** — подставить значения и получить результат
2. **Обратное решение** — найти значение параметра при известном результате
3. **Парсинг size_lookup** — определить порядок столбцов в LookupTable

## Решение

Создать полноценный универсальный парсер формул Revit в `SmartCon.Core`:

1. **Tokenizer** — разбивает строку на токены
2. **Parser** — строит Abstract Syntax Tree (AST)
3. **Evaluator** — вычисляет значение AST при заданных параметрах
4. **Solver** — обратное решение (алгебраическая инверсия или бисекция)

### Поддерживаемый синтаксис

- Арифметика: `+`, `-`, `*`, `/`, `^`, `%`
- Сравнения: `<`, `>`, `<=`, `>=`, `=`, `<>`
- Логика: `if()`, `and()`, `or()`, `not()`
- Тригонометрия: `sin()`, `cos()`, `tan()`, `asin()`, `acos()`, `atan()`
- Математика: `abs()`, `sqrt()`, `round()`, `roundup()`, `rounddown()`, `min()`, `max()`
- Константы: `pi`, `e`
- Единицы: `mm`, `m`, `ft`, `in`

### Стратегия SolveFor

- **Линейные формулы:** алгебраическая инверсия дерева AST
- **Нелинейные (if, trig):** метод бисекции на интервале допустимых значений
- Результат округляется до 6 знаков (decimal feet)

## Последствия

**Плюсы:**
- Универсальный модуль, переиспользуемый в будущих плагинах
- Живёт в Core — тестируется без Revit
- Покрывает любые формулы Revit

**Минусы:**
- Значительный объём реализации (~500-800 строк)
- Обратное решение нелинейных формул — приближённое

## Альтернативы

1. **Перебор типоразмеров:** Не парсить формулы, а перебирать все TypeId пока радиус не совпадёт. Проще, но не работает для параметров экземпляра с формулами.
2. **Внешний парсер (NCalc, Roslyn Scripting):** Дополнительная зависимость, не поддерживает синтаксис Revit (if с 3 аргументами, единицы).
