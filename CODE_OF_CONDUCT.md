# Code of Conduct

Проект SmartCon следует
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Кратко:

- Будьте вежливы и уважительны друг к другу.
- Приветствуются конструктивная критика и обсуждение архитектурных решений.
- Не допускаются личные нападки, троллинг и любая форма дискриминации.
- Обсуждение политики, религии и прочих офф-топик тем — вне скоупа репозитория.

Нарушения можно сообщать через [private security advisory](https://github.com/Alexandrisius/AGK-SmartCon-Pro/security/advisories/new)
или напрямую владельцу репозитория.

Полный текст: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.
